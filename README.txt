Seo Monitor is a free WordPress plugin to monitor your SEO performance for your website(s).

== Description ==

Are you also tired to pay expensive monthly subscriptions to keep track of your keywords. With this plugin you don’t need this services anymore.
Keep track of your keywords on your own WordPress site for free.

The Seo Monitor was built to offer a simple ranking measurement tool for WordPress websites. You can now view how your website (or other websites) ranks for a number of different keywords right within the WordPress admin panel.

= Features =
* Track your rankings on Google
* Search using other Google Country URLs (e.g. google.co.uk)
* Add as many websites and keywords you want
* View historical ranking graph for each keyword
* Keywords can be filtered on website, group, search engine or by search term. All data is sortable.
* Action Log, all actions are logged in the action log. This log will automatically clean items older than two weeks. You can also delete log actions manually.
* Delete single rankings per keyword, so that you can delete for example sample data.

== Installation ==

1. Download the Seo Monitor plugin from WordPress.org.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add your website with all the keywords on the "sites" page.
4. Go to the “keywords” page and the first rankings will be tracked.

== Frequently Asked Questions ==

= Is it possible to add additional search engines like Yahoo, Bing and others? =

Yes this is possible, you can do this yourself with the available hooks and actions. You can see an example in the file admin/searchengine/class-seo-monitor-search-engine-google.php how to do this.
Or you can buy the premium version (not available yet), this version will support Bing and Yahoo.
